Công nghệ sử dụng
- ASP.NET WebForms (.NET Framework)
- C#
- SQL Server
- Bootstrap
- HTML/CSS/JavaScrip

---

Tính năng chính của website

Người dùng
- Đăng ký / đăng nhập
- Xem danh sách tin bất động sản
- Tìm kiếm và lọc theo:
  - loại bất động sản
- Xem chi tiết tin

*Người bán / Chủ tin
- Đăng tin bất động sản
- Chỉnh sửa tin
- Xóa tin
- Quản lý danh sách tin của mình
- Upload hình ảnh

Quản trị viên (Admin)
- Quản lý người dùng
- Quản lý danh mục bất động sản
- Xóa / khóa tài khoản
